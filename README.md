# Replication Code: Misappropriating Vulnerability

Replication materials for the paper "Misappropriating Vulnerability."

## Structure

```
├── R Code/
│   ├── 1_tract level concentrated disadvantage items_260510.R   # Build CDI index (tract level)
│   ├── 2_American Violence data cleaning_260508.R               # Clean & merge violence data (main)
│   ├── 3_Other Figures_260508.R                                 # Generate paper figures
│   └── 4_county level robustness check.R                        # County-level robustness check
│
├── Data/
│   ├── fatal_shootings_2019-2023.RDS     # Raw: police shooting incidents (do not modify)
│   ├── SVI_2020_US_tract.csv             # Raw: CDC SVI — tract level (do not modify)
│   ├── SVI_2020_US_county.csv            # Raw: CDC SVI — county level (do not modify)
│   ├── SVI2020Documentation_08.05.22.pdf # SVI codebook
│   └── (created) ...                     # Output files generated by the scripts below
│
└── Figure/                               # Output figures and maps
```

## How to Run

Execute R scripts in order:

1. `1_...R` — generates `(created) concentrated disadvantage items.RDS`
2. `2_...R` — generates `(created) finaldata_*.RDS/.csv` (tract & county level)
3. `3_...R` — generates figures in `Figure/`
4. `4_...R` — county-level robustness check (supplemental)
