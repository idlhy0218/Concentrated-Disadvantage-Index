About

This repository provides the Concentrated Disadvantage Index (CDI), a composite measure of neighborhood-level socioeconomic vulnerability constructed for use in research on police violence and structural inequality.

The index is built from the CDC Social Vulnerability Index (SVI, 2020) at the census tract level, then aggregated to the county level for robustness checks. It is linked to police fatal shooting data (2019–2023) from the Washington Post database.

The dataset and replication materials accompany the paper "Misappropriating Vulnerability" (under review).

Replication

Run scripts in the following order to reproduce all results and figures.

01

1_tract level concentrated disadvantage items.R

Constructs the CDI from SVI tract-level data. Outputs concentrated disadvantage items.RDS.

02

2_American Violence data cleaning.R

Cleans and merges police shooting data with CDI. Outputs finaldata_*.RDS/.csv.

03

3_Other Figures.R

Generates all paper figures. Outputs to Figure/.

04

4_county level robustness check.R

County-level robustness analysis (supplemental). Stata version also available.

R ≥ 4.0 Stata 17+ tidyverse sf haven
